Objetivo

Crear un web site que les permitirá a los usuarios conocer nuevos destinos turísticos de los cuales podrán disfrutar solos o con sus seres queridos durante su tiempo libre, esto se lograra mediante una página la cual proporcionara una recopilación de consejos, sugerencias y anécdotas de algunos de los lugares más emblemáticos de Antioquia.

Objectivos especificos

● Analizar los requerimientos que tienen los usuarios para escoger sus planes de viaje por medio de la investigación previamente realizada.

● Diseñar un web site teniendo en cuenta el análisis de la información recolectada y que sirva para resolver la problemática encontrada.

● Desarrollar un web site que cumpla con los requisitos propuestos durante el análisis y diseño mediante la tecnología de desarrollo html, css y js y una interacción lógica con el servidor aún por definir.

● Realizar pruebas a las diferentes funcionalidades del sistema, con el fin de encontrar fallos, para cumplir con el objetivo general.
